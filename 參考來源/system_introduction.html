20 個分析模組

涵蓋技術面、籌碼面、基本面、供需面、Smart Money 五大維度

▶ 技術分析模組（A~E）

代號模組流派輸出分數
A 動能共振 RSI / MACD / KD 三指標共振分數 0~3
B VCP 價格收縮 Minervini 收縮等級 + 突破樞紐 0~3
C RS 相對強度 O'Neil 63 / 126 日回報率 + 近高比 0~3
D 布林通道 Bollinger 擠壓 / 突破 / 帶寬百分位 0~2
E 費波那契 Fibonacci 回撤位 / 支撐 / 目標價 0~3

▶ 籌碼、基本面 & 供需模組（F~J）

代號模組資料源輸出分數
F 籌碼集中度 FinMind 法人 法人連買天數 / 集中度標籤 0~4
G Weinstein 階段 MA30 週線 Stage 1~4 + 突破確認 0~3
H Darvas 箱型 價格突破 箱頂/箱底 + 突破訊號 0~3
I 基本面分析 FinMind 財務 營收年增率 / 基本面標籤 0~3
J 供需原理 蕭明道 Mindao S/D 分類 / 角度 / 8MA / 5MV / 進出場 0~5

▶ Smart Money Concepts（SMC 1~4 + CRT + Entry）

代號模組概念輸出分數
SMC1 價格結構 Swing → BOS / CHoCH 趨勢 + 結構突破方向 0~3
SMC2 流動性池 EQH / EQL → BSL / SSL 最近流動性距離 + Sweep 0~3
SMC3 Order Block 機構訂單區塊 牛/熊 OB 區間 + 回測狀態 0~3
SMC4 Fair Value Gap 失衡缺口 + Fib 共振 FVG 區間 + 費波那契共振 0~3
CRT 大K棒區間 TBS / TWS 掃單模型 區間 / 均衡線 / 掃單型別 0~3
Entry SMC 複合進場 3 模型 + Fib50 + K棒確認 方向 / 進場區 / 停損 / 目標 / R:R 0~5

▶ 主力偵測器（DET 1~5）

代號偵測器功能輸出分數
DET1 出貨偵測 高檔滯漲 + 上影線 + 籌碼流出 + RS 衰退 等級 / 高點 / 支撐 0~5
DET2 量價背離分級 三級偵測：警示 → 減碼 → 出場 背離級數 / 計數 0~3
DET3 甩轎偵測 趨勢 + 急跌 + 收回 + 籌碼 + 量能 等級 / 回穩價 0~5
DET4 起漲偵測 築底 → 突破 → 翻多三階段 等級 / 進場 / 停損 / 目標 0~15
DET5 四階段標籤 吸籌 → 拉升 → 洗盤 → 出貨 階段 / 信心度 / 理由 0~5

另有 盤中多頻率蕭明道模組 (INTRA)：60m + 15m 看大做小，日線起漲 → 盤中確認
另有 K 棒形態模組：Pin Bar / Engulfing / Inside Bar 三種經典形態辨識（0~3 分）

01產業急拉買5 min日內漲幅 > 1% + 外盤比率 > 50%

03爆量啟動買5 min累計量 > 均量 3 倍 + 突破近高

06甩轎回穩買30 min盤後甩轎候選 + 盤中回穩確認
07出貨預警賣30 min盤後出貨標記 + 盤中跌破支撐 + 放量
08價量背離賣10 min價創新高但量縮 < 前波高量 60%
09急拉急殺雙向2 min30 秒內漲跌 ≥ 0.7% + 量能脈衝 3 倍
