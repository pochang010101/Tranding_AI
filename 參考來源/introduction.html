  • MA 五線譜 — 5 / 10 / 20 / 60 / 120 日均線
  • RSI 14 — Wilder 平滑法，超買/超賣判定
  • MACD — DIF / MACD / 柱狀體（12, 26, 9）
  • 布林通道 — 上/中/下軌 + 帶寬

AI 機器學習引擎

從特徵工程到模型訓練，內建防未來函數保護。

  • 20+ 特徵 — 報酬率、乖離、動能、量比、籌碼

  • RandomForest — 300 棵樹、balanced 權重
  • 時間切分 — 前 70% / 後 30%，嚴禁洗牌
  • 特徵重要度 — 自動排序、視覺化
🧪

回測引擎

向量化回測含台股實際交易成本，避免高估績效。

  • 訊號延遲 — signal.shift(1)，前日決策今日生效
  • 成本模型 — 手續費 + 證交稅 + 滑價
  • 績效指標 — 年化報酬、夏普值、MDD、勝率
  • 資金曲線 — 含買賣點標記的視覺化圖表
  • 成本僅計進出 — 持倉不動日無額外成本

  • 大盤環境評估（趨勢、動能）

  • 布林通道突破
  • MA 黃金/死亡交叉

  • 模型買進機率計算
  • 資金流向評分排序

  • 可選 Email 通知

StockAnalyzer 一行呼叫

統一門面類別，支援鏈式呼叫與逐步操作。

from stock_ai.config import CONFIG
from stock_ai.analyzer import StockAnalyzer

# 一鍵跑完整流程
result = StockAnalyzer(CONFIG).run()

# 或逐步操作（便於除錯）
analyzer = StockAnalyzer(CONFIG)
analyzer.load()              # 抓資料
analyzer.add_indicators()    # 套指標
analyzer.build_features()    # 建特徵
analyzer.train()             # 訓練模型
analyzer.backtest()          # 含成本回測
analyzer.latest_signals()    # 最新買賣建議

防坑設計 & 獨特優勢

針對量化回測常見陷阱，內建多重防護機制。

防未來函數三層檢查

特徵只用當日以前資料、公告日對齊偏移 90 天、assert_no_lookahead 自動斷言驗證。

訊號延遲一日

signal.shift(1) × 當日報酬：前日收盤決策，今日才生效，杜絕偷看當日資訊。

時間序列嚴格切分

前 70% 訓練、後 30% 測試，嚴禁隨機洗牌，真實反映上線後表現。

台股成本最精確

手續費 + 證交稅 + 滑價完整計入，一進一出 0.685%，避免績效灌水。

dry_run 安全預設

Email 通知預設只印不寄（dry_run=True），避免誤觸發批量郵件。

金鑰零寫死

所有 API Token、Gmail 密碼一律由環境變數讀取，原始碼中不留任何敏感資訊。

模型 × 資金交叉驗證

模型買進但資金流出 → 不建議買進，雙重確認降低虛假訊號。

純看盤不下單

Gradio 介面提供分析與看盤，不涉及實際下單，使用安全無風險。
