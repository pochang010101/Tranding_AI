Strategy System

22 個日 K 策略

六大系列覆蓋隔日沖、波段突破、扣抵共振、型態識別、技術指標、空頭做空，全方位多空佈局。

O 系列 — 隔日沖（T+1 極短線）

代碼策略名稱核心邏輯停損 / 停利
O-1凹洞量後攻擊底部縮量形成凹洞 → 放量漲升啟動-1.5% / +3%
O-2量增突破前高量增 ≥1.5x + 突破 13 日新高-1.5% / +3%
O-5縮量整理末端攻擊3 日量縮整理 + 突破近 3 日高點-1.5% / +3%

S 系列 — 波段突破（3~10 日）

代碼策略名稱核心邏輯停損 / 停利
S-1三盤突破連續 3 日突破前高，多方續攻-5% / +10%
S-2龍回頭上升趨勢回檔至均線後再起攻-5% / +12%
S-3飄（量縮創高）量縮卻創新高 + 漲幅 <5%，惜售格局-5% / +15%
S-4均量帶動上揚5MV 金叉 13MV，量能齊揚確認-5% / +10%
S-5均線金叉扣低8MA 金叉 21MA + 扣抵值低-5% / +10%
S-6盤整突破區間整理 5 日 → 2.0x 放量突破-5% / +12%
S-7攻守角度比前段陡攻 → 淺回檔 → 重新轉強-5% / +12%

K 系列 — 扣抵共振（林教授流派）

代碼策略名稱核心邏輯停損 / 停利
K-1三層均線同步扣低8MA、21MA、55MA 同時站上前收，三線扣低共振-5% / +12%
K-2三層均量同步扣低5MV、13MV、34MV 同時量增，量能三線扣低-5% / +15%
K-3扣抵雙共振（金級）K-1 + K-2 同時成立，極低頻率的金級訊號-5% / +20%

P 系列 — 型態面

代碼策略名稱核心邏輯停損 / 停利
P-1W 底突破頸線兩個相近低點 + 量增突破頸線-5% / +12%
P-2旗型整理突破急漲 → 量縮平台整理 → 帶量突破-5% / +12%
P-3三角收斂突破高點降低 + 低點升高 → 突破上沿-5% / +10%
P-4布林收斂突破布林帶寬 <1%（極度收斂） + 放量突破-5% / +12%
P-5前波高之上突破平台位置 > 前波高 + 帶量突破-5% / +14%

T 系列 — 技術指標

代碼策略名稱核心邏輯停損 / 停利
T-1季線翻揚轉強55MA 由下向上走平 + 價格站穩季線之上-5% / +15%
T-2MACD 紅柱擴張MACD > Signal 且柱狀體持續擴大-5% / +12%

SD 系列 — 空頭做空

代碼策略名稱核心邏輯用途
SD-1凹峰量爆量殺高檔量爆但價格下跌（供給傾瀉）賣出警示
SD-2量增價跌空頭量增趨勢但價格持續下跌空頭確認
SD-3三盤跌破續空連續 3 日跌破前低空方續攻
SD-4三層均線扣高三層均線同步走低（空頭共振）系統性空頭
Intraday Signals

盤中六訊號

5 分鐘 K 棒即時觸發，三組多空配對（B1/S1、B2/S2、B3/S3），內建漲跌停互斥與 B2/S2 衝突處理邏輯。

買進訊號（多頭）

B1

突破整理上沿

收盤價突破近 20 根最高價 + 量增 ≥1.8x 均量。多方啟動，漲停鎖住時不觸發。

B2

惜售守住支撐

守住 MA20 支撐 + 量縮 ≤0.5x（惜售）。需當根未下跌（close ≥ prev），與 S2 互斥。

B3

洗盤後翻越

跌破 MA20 後強勢收回 + 大量 ≥1.8x。假跌破真洗盤，多方反撲。

賣出訊號（空頭）

S1

跌破整理下沿

收盤價跌破近 20 根最低價 + 量增 ≥1.8x。空方啟動，跌停鎖住時不觸發。

S2

止漲動能衰竭

量縮 ≤0.5x 不見新高（止漲）。需當根下跌（close < prev），排除漲停鎖住與 MA20 支撐。

S3

跌破均線支撐

收盤跌破 MA20 + 量增 ≥1.8x。均線支撐失守，趨勢反轉確認。

互斥規則

B2 / S2 衝突處理 — 守住 MA20 支撐 → 歸 B2，S2 不觸發
漲停鎖住 — B1 不觸發（掛單排隊非真突破）、S2 不觸發
跌停鎖住 — S1 不觸發
開盤初期 — 需至少 lookback_bars/2 根 K 棒，前幾根不觸發
Multi-dimensional Fusion

四大模組融合計分

將三大流派的策略分門歸類為四大模組，各自評分後加上籌碼、產業、角度等多維加分，形成最終共振評分。

Module L — 飆股 DNA
蕭明道流派
O-1、O-2、O-5、S-3、S-6
量價供需四象限 + 三均線多排，短線買點。
Module L4 — 觔斗雲助漲
林教授流派
K-1、K-2、K-3、S-5
費氏三層均線/均量扣抵共振，波段超前指標。
Module C — 強勢突破集中
廖崧沂流派
S-1、S-2、S-4、T-1、T-2
RS ≥80 百分位 + 產業輪動，選最強族群中最強股。
Module F — 底部布局
反向策略
P-1、P-4
W 底 / 布林極度收斂，底部磨底期反彈伏兵。

綜合評分公式

總分 = 模組分（L / L4 / C / F 各策略命中加分）
     + 量價後處理（量增價漲 +2 ∼ 量縮價跌 -1）
     + 扣抵共振（三層 MA 同步扣低 +3、三層 MV 同步扣低 +2）
     + 角度企圖（陡峭 +2、緩漲 +1、陡跌 -2）
     + 籌碼面（外資/融資強度 +1 ~ +3）
     + 產業 RS（相對強度 +1 ~ +3、弱勢 -1）
     + 多維共振（2 維 +2、3 維 +4、4 維 +7）

方向判定：總分 ≥4 + 至少 1 個日 K 訊號 → 多
              總分 ≤-4 + 至少 1 個空頭訊號 → 空
              其餘 → 中性

Core Philosophy

核心理念

量價為王

成交量是價格的靈魂。先看量再看價，量是因、價是果。

均線不騙人

費氏數列三層均線的扣抵狀態是最強超前指標，領先價格 3~7 天。

籌碼藏真理

外資/融資動向往往領先技術面，籌碼集中度揭示主力意圖。

環境要順勢

強多環境積極進場，弱空環境停止新開倉，環境閘門決定策略權重。

紀律第一

硬性停損 -7% 絕無例外。分批停利（+8% 出五成、+10% 出兩成五），讓利潤奔跑。

多維共振

量價、扣抵、籌碼、產業 RS 四維度同時確認，共振越多、勝率越高。
